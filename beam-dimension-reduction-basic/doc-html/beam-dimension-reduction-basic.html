Abstract

Using the assumption of cylindrical invariance, we derive a one-dimensional beam model representing a prismatic elastic body. In this elementary presentation, we limit attention to linear elasticity and neglect any gradient effect (i.e., we derive the leading-order contributions to the 1D energy). The approach works for arbitrary shapes of cross-section, arbitrary material symmetries and constitutive laws, and does not require the elastic properties to be homogeneous in the cross-section. It does not make any ad hoc kinematic assumption such as unshearability.
